Les icônes suivantes sont nécessaires pour l'extension :
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

Vous pouvez créer ces icônes en utilisant un outil de création d'images comme Photoshop, GIMP, ou un service en ligne.

Placez les icônes dans ce dossier (public/icons/) pour que l'extension fonctionne correctement.
